<details>
<summary> Two Pointers</summary>

### Concept
Used when processing elements from both ends or adjusting positions efficiently.

### Problems
- Two Sum II - Input Array Is Sorted
- Remove Duplicates from Sorted Array
- Move Zeroes
- Squares of a Sorted Array
- Container With Most Water
- 3Sum

### Key Learning
- Left and right pointer movement
- Shrinking and expanding boundaries
- Avoiding nested loops

</details>

<details>
<summary> Sliding Window</summary>

### Concept
Used for subarray/substring problems where a window moves dynamically.

### Problems
- Maximum Average Subarray I
- Longest Substring Without Repeating Characters
- Longest Repeating Character Replacement
- Minimum Window Substring
- Permutation in String

### Key Learning
- Fixed vs variable window
- Expanding and shrinking window
- Maintaining frequency maps

</details>

<details>
<summary> Prefix Sum</summary>

### Concept
Precompute cumulative sums to answer range queries efficiently.

### Problems
- Running Sum of 1D Array
- Subarray Sum Equals K
- Find Pivot Index
- Range Sum Query - Immutable

### Key Learning
- Precomputation
- Range queries in O(1)
- Space-time tradeoff

</details>

<details>
<summary> Hashing / Frequency</summary>

### Concept
Use hash maps or arrays to store frequency and enable fast lookup.

### Problems
- Two Sum
- First Unique Character in a String
- Valid Anagram
- Ransom Note
- Group Anagrams
- Top K Frequent Elements

### Key Learning
- Frequency counting
- Fast lookup
- Trade space for speed

</details>

### Problems

- Maximum Product Subarray

### Key Learning

- Handling negatives
- Greedy decisions
