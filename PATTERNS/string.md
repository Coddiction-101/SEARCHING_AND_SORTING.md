# STRING PATTERNS PLAYBOOK

A structured collection of string problems organized by core DSA patterns.
Focus is on clarity, pattern recognition, and efficient manipulation.

---

## INDEX

- Two Pointers
- Sliding Window
- Hashing / Frequency
- Stack Based
- Basic String Manipulation
- Advanced Patterns

---

<details>
<summary> Two Pointers</summary>

### Concept
Used when comparing or modifying characters from both ends or traversing efficiently.

### Problems
- Valid Palindrome
- Reverse String
- Reverse Words in a String
- Valid Palindrome II

### Key Learning
- Left and right pointer traversal
- Skipping characters
- In-place modifications

</details>

---

<details>
<summary> Sliding Window</summary>

### Concept
Used for substring problems where a window expands and shrinks dynamically.

### Problems
- Longest Substring Without Repeating Characters
- Longest Repeating Character Replacement
- Minimum Window Substring
- Permutation in String
- Find All Anagrams in a String

### Key Learning
- Variable window size
- Frequency tracking
- Maintaining valid window conditions

</details>

---

<details>
<summary> Hashing / Frequency</summary>

### Concept
Store character counts for quick lookup and comparisons.

### Problems
- Valid Anagram
- Ransom Note
- First Unique Character in a String
- Group Anagrams
- Isomorphic Strings

### Key Learning
- Frequency arrays vs hash maps
- Character mapping
- Efficient comparisons

</details>

---

<details>
<summary> Stack Based</summary>

### Concept
Used when order and pairing of characters matter.

### Problems
- Valid Parentheses
- Remove All Adjacent Duplicates In String
- Decode String

### Key Learning
- LIFO structure
- Matching patterns
- Nested processing

</details>

---

<details>
<summary> Basic String Manipulation</summary>

### Concept
Direct operations on strings without advanced patterns.

### Problems
- Longest Common Prefix
- Length of Last Word
- Count and Say

### Key Learning
- Traversal and parsing
- String building
- Edge case handling

</details>

---

<details>
<summary> Advanced Patterns</summary>

### Concept
More complex string processing techniques used in higher-level problems.

### Problems
- KMP Algorithm (Pattern Searching)
- Rabin-Karp Algorithm
- Z Algorithm
- String Compression
- Reorganize String

### Key Learning
- Pattern matching optimization
- Rolling hash
- Preprocessing techniques

</details>

---

## NOTES

- Strings are often disguised array problems
- Focus on patterns like sliding window and hashing
- Always handle edge cases (empty string, single char)
- Practice both brute force and optimized approaches

---

## PROGRESSION STRATEGY

1. Start with basic manipulation
2. Move to two pointers
3. Learn sliding window deeply
4. Practice hashing problems
5. Explore advanced algorithms

---

## GOAL

Develop strong intuition for substring problems and efficient string transformations.
